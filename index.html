Testes de Sensores

1️⃣ Registrar Leitura de Sensores

Simula envio de dados do ESP32

Exemplo de Request:
POST /backend/api/sensores.php?acao=registrar HTTP/1.1
Authorization: Bearer chave_api_esp32_segura
Content-Type: application/json

{
"usuario_id": 1,
"temperatura": 28.5,
"umidade_ar": 65,
"umidade_solo": 45
}

2️⃣ Obter Última Leitura

Recupera a leitura mais recente dos sensores

Exemplo de Request:
GET /backend/api/sensores.php?acao=ultima_leitura HTTP/1.1
Cookie: PHPSESSID=...

3️⃣ Obter Histórico de Leituras

Recupera leituras dos últimos dias

4️⃣ Obter Estatísticas

Calcula média, máxima e mínima dos dados

Content-Type: application/json
Cookie: PHPSESSID=...

"duracao_minutos": 10,
"tipo_comando": "manual"

2️⃣ Desligar Irrigação

4️⃣ Histórico de Irrigações

Lista todos os acionamentos
